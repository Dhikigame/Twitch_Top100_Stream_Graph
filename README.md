# Twitch_Top100_Stream_Graph

streamdata_insert.phpでデータベースにトップ100の視聴数のTwitch情報を格納する。

streamdata_stats_graph.pyで指定したIDと日時範囲を指定することで配信ごとの統計とグラフを生成する。


# Requirement
* Python3
* PHP7
* MySQL
